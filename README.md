# rust-projects
various rust projects

## install rust
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
rustc --version
rustup update
cargo --version
```

## compile
```
rustc hello.r
```

## various build commands
```
cargo new project
cargo build
cargo build --release
cargo run
cargo test
```
